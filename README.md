## ✨ Features

- 🎨 **Modern UI/UX** - Clean, professional design with glassmorphism effects
- ✨ **Smooth Animations** - Powered by Framer Motion for engaging user experience
- 🖱️ **Custom Cursor** - Interactive cursor that responds to hover states
- 🌌 **Particle Background** - Dynamic animated particles using tsParticles
- ⌨️ **Typing Animation** - Animated role titles in hero section
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
- 🎯 **Skill Filtering** - Filter skills by Development or DevOps categories
- 🗂️ **Project Showcase** - Filterable project cards with hover effects
- 📜 **Timeline Experience** - Animated work experience timeline
- 📧 **Contact Form** - Easy way to get in touch

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v16 or higher)
- **npm** or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/premgowda7/prema-portfolio.git
   ```

2. **Navigate to project directory**
   ```bash
   cd prema-portfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   ```
   http://localhost:3000
   ```

### Build for Production

```bash
npm run build
```

The production files will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```
